Example facilitator plan

First pilot session plan.

A 45-60 minute example session for a small pilot group. It keeps the first powered build narrow: recognise safe power, sort unplugged controls, then light one LED module after an adult check.

Length
45-60 minutes
Children
4-6
Adults
1 facilitator, 1 helper recommended
Code
No child-authored code

Timing

The session has one clear safety path.

The first half builds language and confidence without powered circuits. The powered activity starts only after the adult has checked polarity, path and switch state.

  1. 0-5 Open the room rules

    No power is connected until an adult checks the path, polarity and switch state.

  2. 5-15 Recognise safe power

    Children practise positive, negative and count-back language with adult-controlled cells.

  3. 15-25 Sort controls

    Buttons, switches, knobs and joysticks stay unplugged while children sort by user action.

  4. 25-45 Power one LED load

    Pairs build the LED-module circuit with the pack switched off, then switch on briefly after adult check.

  5. 45-60 Reset and count back

    Switch off, unclip leads, count all power items back and record what needed more explanation.

Cards used

Three Power Cards are enough for the first pilot.

The plan deliberately avoids motors, LED strips and preloaded boards until the basic safety routine is calm and repeatable.

Level 1

Recognise safe power sources

I can recognise common cells, packs, polarity marks and safe handling rules.

Prompt
Which marks show positive and negative?
Stop
Children can name polarity and one safety limit.
Level 1

Recognise input controls

I can recognise buttons, switches, knobs and joysticks as input controls.

Prompt
Which control do you press, switch, turn or steer?
Stop
Children sort controls by physical action.
Level 2

Power one load

I can power one safe load with the right polarity.

Prompt
How can we make one complete path without crossing positive and negative?
Stop
Each pair briefly lights the LED, switches off and resets.

Setup

Adult preparation is part of the lesson.

  • Print the safe-circuit diagram before children arrive.
  • Lay out power recognition, unplugged controls and powered-circuit areas separately.
  • Prepare AA packs switched off before handout.
  • Keep loose AA cells and coin cell in adult-controlled reach only.
  • Check clip lead insulation and LED polarity labels.
  • Keep the optional motor load out of the first pilot tray.

Safety

Stop before adding more challenge.

  • No power connected until an adult checks polarity, circuit path and switch state.
  • Use the LED module only for the first powered circuit.
  • Keep loose cells under adult control and count them back.
  • Switch battery packs off before handout, checking, reset or pack-away.
  • Children handle only the prepared low-voltage parts named for the active card.
  • Stop once every pair has briefly lit the LED, switched off and reset.

Reset

Pack-away is evidence, not admin.

The pilot should record whether the materials, wording and adult ratio worked in the room. Count-back time matters because it affects whether the session can scale.

Reset checklist

  • Switch off every AA pack before leads are removed.
  • Unclip and untangle all clip leads.
  • Return LED modules unplugged to the power load compartment.
  • Return controls dry and unplugged, with connector pins checked.
  • Count loose cells, coin cell, AA packs and clip leads before closing the kit.
  • Record warm wires, damaged insulation, missing labels or confusing prompts.

Defer until kit is confirmed

  • Motors and moving shafts.
  • LED strips and current-limit checks.
  • Preloaded boards, displays, servos and sensor indicators.
  • Separate structures and sensing stations.

This is a pilot plan, not a finished public timetable.

It shows how the first session can stay structured, safe and small enough to learn from.
